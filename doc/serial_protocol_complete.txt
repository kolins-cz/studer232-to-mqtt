1. Appendix

The information in the appendices is valid for the latest software release. For a detailed changelog, please see the document
Release_Rxxx.pdf in the latest “Studer system update” available on www.studer-innotec.com.

A major release (from R5xx to R6xx, for example) can imply significant changes that should be validated.
